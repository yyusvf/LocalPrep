LocalPrep

v0.1.0 · Windows 10 / 11

Desktop tool for preparing local audio files — convert formats & sample rates, edit metadata, reorder tracklists. Built for anyone who uses local files in Spotify, Apple Music, or Tidal.

Download for Windows GitHub
No cloud No subscriptions No telemetry ffmpeg bundled Auto-updates

Features

Sample Rate Conversion

Batch-convert to 44.1 kHz, 48 kHz, 96 kHz, or any target rate. Overwrite or keep originals.

Format Conversion

Convert between MP3, FLAC, WAV, OGG, M4A, AAC, AIFF with configurable quality settings.

Metadata Editor

Edit ID3/Vorbis tags, embed cover art, batch rename with token patterns, set track numbers.

Tracklist Sequencer

Drag & drop songs into order across multiple discs. Writes track and disc numbers automatically.

Audio Preview

Built-in player with volume control. Play any file with spacebar or right-click.

Undo System

Every conversion and rename is reversible. Full history tab with one-click undo.

Also included

Drag & drop folders directly onto any tab
Right-click any audio file in Windows Explorer → LocalPrep context menu (no admin needed)
Sortable, resizable columns — state persists between sessions
Auto-updates via GitHub Releases — silent background check on startup
English & German — easily extendable to more languages

Supported formats

.mp3 .flac .wav .ogg .m4a .aac .aiff .aif

Free to download

No account needed. No cloud. Installs per-user with no admin rights required.
